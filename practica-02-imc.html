Checklist rapido

Concepto clave: formula del IMC

El IMC se calcula con la formula:

IMC = peso(kg) / (altura(m) ** 2)

Clasificacion basica

Rango IMC Categoria
Menor a 18.5 Bajo peso
18.5 a 24.9 Normal
25.0 a 29.9 Sobrepeso
30.0 o mas Obesidad
Tip: redondea el IMC a 2 decimales con round(valor, 2).

Guia de trabajo (paso a paso)

  1. Crea dos TextField: peso (kg) y altura (m).
  2. Agrega un boton Calcular IMC.
  3. Al hacer click, convierte los valores a float.
  4. Calcula el IMC y muestra resultado y categoria.
  5. Valida que los campos no esten vacios y sean positivos.
import flet as ft

def main(page: ft.Page):
    peso = ft.TextField(label="Peso (kg)")
    altura = ft.TextField(label="Altura (m)")
    resultado = ft.Text("")
    categoria = ft.Text("")

    def calcular(e):
        try:
            p = float(peso.value)
            a = float(altura.value)
            if p <= 0 or a <= 0:
                raise ValueError
            imc = round(p / (a ** 2), 2)
            resultado.value = f"IMC: {imc}"
            if imc < 18.5:
                categoria.value = "Categoria: Bajo peso"
            elif imc < 25:
                categoria.value = "Categoria: Normal"
            elif imc < 30:
                categoria.value = "Categoria: Sobrepeso"
            else:
                categoria.value = "Categoria: Obesidad"
        except ValueError:
            resultado.value = "Datos invalidos. Usa numeros positivos."
            categoria.value = ""
        page.update()

    page.add(
        peso,
        altura,
        ft.ElevatedButton("Calcular IMC", on_click=calcular),
        resultado,
        categoria
    )

ft.run(main)
